# Contributing

Спасибо за интерес к проекту. Цель репозитория — поддерживать маленькую, предсказуемую утилиту для сборки Markdown в PDF A4 с русскоязычной учебной и проектной документацией в фокусе.

## Локальная разработка

```bash
python3 -m venv .venv
. .venv/bin/activate
python -m pip install --upgrade pip
python -m pip install -e ".[dev]"
npm install
```

## Проверки перед pull request

```bash
python -m ruff check .
python -m pytest -q
npm run pdf
npm run example
```

Если изменение затрагивает рендер Mermaid, проверьте сценарий без доступного браузера: PDF должен собираться через встроенный рендерер для поддержанных типов диаграмм.

## Стиль изменений

- Держите изменения небольшими и связанными с одной задачей.
- Добавляйте тесты для исправленных багов и новых веток поведения.
- Не коммитьте локальные окружения, кеши и временные файлы.
- Сгенерированные PDF можно обновлять, если изменился видимый результат примеров или документации.
